## Purpose
To provide Mosaic with structured reference material on visual, narrative, product, and conceptual systems. This canvas gives Mosaic a foundation to interpret, generate, and evaluate work based on design thinking, storytelling, UX logic, and brand systems.

## Usage Conditions
- Used to extend Mosaic’s system-level reasoning
- Acts as a reference layer — not meant for client delivery
- Can be updated with evolving agency frameworks or best practices

## Knowledge Domains Included

### 1. Visual Design Styles
- Editorial / Swiss / Maximalist / Bauhaus / Minimalist / Brutalist / Humanistic / Experimental
- References mapped to common layout logic, color language, typography, white space principles

### 2. UX/UI Principles
- Heuristic checks (Nielsen, Krug, etc.)
- Accessibility considerations
- Component logic (cards, grids, navs)
- Interaction models (mobile-first, adaptive, scroll logic)

### 3. Narrative Structures
- Brand storytelling frameworks (Golden Circle, Hero’s Journey)
- Archetype language and emotional drivers
- Tone ladders and voice grids

### 4. Concepting Approaches
- Lateral thinking vs. vertical logic
- Moodboard anchoring and conceptual triggers
- “What if?” matrix creation

### 5. Digital Product Thinking
- Value propositions mapped to flows
- Entry point → core task → reward loop logic
- Microcopy and affordance planning

### 6. Campaign & Performance Layers
- Awareness → Engagement → Conversion
- CTA tuning and scroll-depth strategy
- Post-campaign reflection triggers (via `loop.reflect.canvas`)

## Method
This guide enables Mosaic to:
- Interpret prompts with stylistic logic
- Provide reasoning for design decisions
- Choose appropriate narrative and aesthetic approaches
- Map deliverables to structured experiences
